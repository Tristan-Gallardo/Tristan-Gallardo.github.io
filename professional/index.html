Challenge

The 3 November 2025, I was ask to develop an automatic update system.

Indeed, my supervisor, after seeing some security problem if a user downloaded it themself manually, ask if it was possible to add an automatic update system.

So I tasked myself to implement it and make sure that user cannot use the extension if not updated.

After some researched, I found out that you can execute code outside of the program, by creating a temporary file with the updating code inside, and execute it when leaving the application.

As a result, the user can now update automatically the programs, with mutiple security verifying if the program is correctly closed and not locked.
